ğŸ§© What is Docker Compose?

Docker Compose is a tool that helps me run and manage multi-container
Docker applications easily.
Instead of starting each container manually with long 'docker run'
commands, I can define everything
(services, networks, volumes) inside a single 'docker-compose.yml' file.
Then I can spin up the entire
application stack with a single command.

-----------------------------------------
âš™ï¸ Why do I use Docker Compose?

- Makes it simple to run multiple containers together (like a web app and a
database)
- Keeps environments consistent across development, testing, and production
- Supports environment variables, custom networks, and volumes
- Saves time by managing the entire stack from one YAML file

ğŸ§  How it Works

- 'web' runs an Nginx container serving local HTML files on port 8080.
- 'db' runs a MySQL container with persistent storage and environment
variables.
- Both containers automatically connect over the same network created by
Docker Compose.

To start everything:
    docker compose up -d

To stop and clean up:
    docker compose down
